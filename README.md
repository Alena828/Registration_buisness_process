# Registration_buisness_process

Данный репозиторий содержит наработки по проектированию бизнесс-процесса регистрации собаки в едином Реестре Ленинградской области. Данный процесс проектировался в качестве учебного задания по дисциплине <<Технология проектирования приложений>>.

1. Описание предметной области и процесс.pdf - краткое описание предметной области мониторинга инфорации о бездомных и домашних собаках и непосредственно описание процесса регистрации собаки согласно постановлению правительства Ленинградской области от 15 апреля 2024 год №242 "Об утверждении порядка регистрациимдомашних животных на территории Ленинградской области"

## Use_cases_diagram 

Данный раздел содержит Use-cases диаграммы для процеесса регистрации собаки в Реестре с участием следующих акторов: Заявитель, КОнсультант, Сотрудник реестра, Администратор вет.клиники и Ветеринар. Каждый уровень диаграммы представлен отдельной диаграммой.


## BPMN_diagrams

Данный раздел содержит BPMN-диаграммы следующих процессов: 
   1. Запись на прием для регистрации с участием Заявителя, Консультанта и Администратора вет.клиники
   2. Запись на прием для регистрации с участием Заявителя и Администратора вет.клиники
   3. Фрагмент процесс записи на регистрацию, оканчивающийся на моменте получения персональных данных заявителя. Данный фрагмент используется для разработки экранных форм.

Исходный бизнес--процесс был укорочен для учебных целей.
