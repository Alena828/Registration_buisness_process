# Screen_form 

Данный раздел содержит граф экранных форм для фрагмента бизнесс-процесса регистрации и сами экранные формы, спроектированные в Figmа.

1. Фрагмент процесс.svg - фрагмент процесса записи на регистрацию, для которого разрабтаывались экранные формы.
2. Граф_экранных_форм.png - граф экранных форм, используемых во фрагменте процесса записи.

## sf1.png - Форма записи на регистрацию 
   Заполняется заявителем.
   ### Кнопка <<Отмена>> - прекращение процесса
   ### Кнопка <<Отправить заявку на регистрацию>> - переход  к экранной форме 2.1

## sf2.1.png - Форма первичной обработки заявки на запись 
   Заполняется администратором.
   ### Кнопка << Перейти к формированию заявления на регистрацию >> - переход к форме 4
   ### Кнопка <<Отправить отказ в обработке>> - не активна
   ### Поле <<Отказать в обработке>> - переход к форме 2.2
