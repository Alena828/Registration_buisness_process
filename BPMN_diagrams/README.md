## BPMN_diagrams

Данный раздел содержит BPMN-диаграммы следующих процессов: 
   1. Полная версия процесса записи.svg - Запись на прием для регистрации с участием Заявителя, Консультанта и Администратора вет.клиники
   2. Полная версия процесса с двумя ролями.svg - Запись на прием для регистрации с участием Заявителя и Администратора вет.клиники
   3. Фрагмент процесса.svg - Фрагмент процесс записи на регистрацию, оканчивающийся на моменте получения персональных данных заявителя. Данный фрагмент используется для разработки экранных форм.

Исходный бизнес-процесс был укорочен для учебных целей. Все BPMN-диаграммы составлялись с применением сайта stormbpmn.com.
